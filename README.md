# media

Repositório de **imagens, ícones e logos** para uso **linkado** em projetos (README, sites, documentação, etc.).

## Estrutura sugerida

Organize os arquivos por tipo/projeto, por exemplo:

- `banners/`
- `iconsDvz/`
- `iconsVnuv/`
- `thumbnails/`
- `logos/` (se quiser adicionar depois)

## Como usar (link direto / raw)

Use o formato abaixo:


Exemplo:

```md
![VNUV Icon](https://raw.githubusercontent.com/edsonvazlopes/media/main/iconsVnuv/vnuv2.ico)
